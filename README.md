## Building with CMake

To build the project, follow these steps:

1. Create a build directory:
   ```bash
   mkdir build
   cd build
   ```
2. Run CMake to configure the project:
   ```bash
   cmake ..
   ```
3. Build the project:
   ```bash
   cmake --build .
   ```

Make sure you have CMake installed on your system.
